This project presents an interactive Power BI dashboard analyzing Metro government budget vs actual spending. The goal is to evaluate financial performance, identify overspending or underspending, and provide insights at both summary and detailed levels.
Link - https://app.powerbi.com/links/j_-VMXNP2F?ctid=101da587-1843-4f52-8b8a-17b069c66d33&pbi_source=linkShare&bookmarkGuid=7950b7b6-4b83-4883-bd4f-aaa2309d3cd4
🎯 Objectives
Compare Actual vs Budgeted spending
Identify over/under budget performance
Analyze department and fund-level spending
Provide interactive drillthrough analysis
📈 Key KPIs
Total Actual – Total amount spent
Total Budget – Planned budget
Variance – Difference between actual and budget
Utilization % – Percentage of budget used
Over/Under Budget % – Variance as a percentage of budget
🧠 Key Features
📌 KPI indicators with arrows for quick insights
🎨 Conditional formatting (traffic light system for utilization)
🔎 Drillthrough functionality for detailed analysis
🔘 Navigation buttons between pages
📊 Interactive visuals (bar charts, line charts, donut charts, matrix)
📄 Dashboard Pages
1️⃣ Overview Page
High-level summary of financial performance
KPI cards and trend analysis
Department and fund-level visualizations
2️⃣ Detailed Analysis Page
Drillthrough-enabled detailed view
Department, fund, and business unit breakdown
Variance analysis and comparison
🎨 Design Approach

The dashboard uses a Minion-inspired color theme with a Lego-style layout to create a visually engaging yet structured design. The goal was to balance creativity with clarity and usability.
